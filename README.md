# Care Act Decision Flowchart

This is a simple web application that guides users through a Care Act decision flowchart. It helps determine whether to instruct an advocate based on a series of yes/no questions.

## How to Use

1. Open the `index.html` file in a web browser.
2. The first question of the flowchart will be displayed.
3. Click "Yes" or "No" to answer the question.
4. The next question will be displayed based on your answer.
5. You can go back to the previous question by clicking the "Back" button.
6. The flowchart will guide you to a final outcome: "Instruct" or "Don't Instruct".
